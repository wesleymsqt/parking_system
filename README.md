# 🚗 Sistema de Estacionamento em Tempo Real

## 📌 Descrição
Este projeto consiste em um **sistema de estacionamento inteligente** que monitora em **tempo real** a disponibilidade de vagas. Ele utiliza sensores para detectar a presença de veículos e exibe as informações em uma interface web intuitiva.

## 🛠️ Tecnologias Utilizadas
- HTML: Estrutura da interface web.
- CSS: Estilização da interface.
- Arduino (.ino): Controle dos sensores e comunicação com a interface.

## 🚀 Funcionalidades
- Detecção automática de veículos.
- Atualização **em tempo real** do status das vagas.
- Interface web para monitoramento prático.

## Como Usar
1️⃣ Carregar o código `.ino` no Arduino.
2️⃣ Conectar os sensores corretamente.
3️⃣ Executar a interface web localmente para visualizar o status do estacionamento.

## 📋 Requisitos
🔹 Placa Arduino. (ex: Uno, Nano, etc.)
🔹 Sensores de estacionamento. (ex: ultrassônicos ou infravermelhos)
🔹 Módulo de comunicação. (Wi-Fi ou Bluetooth)
